There are two versions of the program that runs in Windows and Linux.

Program #1: Runs on Windows; Only captures information in Application Layer, Transport Layer and Network Layer.
Program #2: Runs on Linux; Captures the Link Layer.


To execute Program #1 (Windows):
	(Assumption: Python is already added as a PATH variable)
	1. Run Command Prompt with Administrator privileges
	2. Browse to the root directory.
	3. Type "python server.py" and hit return key.
	4. Open a browser window (preferably Chrome) and go to the following URL: http://localhost:3000/

To execute Program #2 (Linux):
	1. Type "python cgi-bin/Sniffer-linux.py"
